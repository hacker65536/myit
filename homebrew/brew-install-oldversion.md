# brew install package (old version)


```
$ brew info PACKAGE
```


if it has multiple versions

```
$ brew switch PACKAGE 0.10.0
```

if there is only one version
```
$ cd /usr/local/Homebrew/Library/Taps/homebrew/homebrew-core/Formula
```
```
$ git log PACKAGE.rb
```

```
$ git checkout <HASH>
```

```
$ brew unlink PACKAGE
```

```
$ brew install PACAGE
```

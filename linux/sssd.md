# ldap_sssd
sssd+ldapの設定

uname -a

install pkg
----
```bash
yum install -y sssd openldap-clients sssd-ldap
```

backup
----
```bash
authconfig --savebackup=2016-07-05
```

resotre
---
```bash
authconfig --restorebackup=2016-07-05
```

authconfig
---
```bash
authconfig \
--enablesssd \
--enablesssdauth \
--enablelocauthorize \
--enableldap \
--enableldapauth \
--disableldaptls \
--ldapserver=ldaps://ldapserver \
--ldapbasedn=dc=company,dc=local \
--enablemkhomedir \
--update
```

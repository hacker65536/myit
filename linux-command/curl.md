
download file


`-O` --remote-name

```bash
curl -O http://fileuri
```

`-o` --output

```bash
curl http://fileuri -o file1 
```

`-s` --silent
```bash
curl -s http://fileuri
```


`-S` --show-error
```bash
curl -S http://fileuri
```

`-L` --location
```
curl -L http://fileuri-redirect
```


```bash
# 
curl -sSLO http://fileuri
```

# vim

## option

-p[N] 
- Open N tab pages. 

-R
- Read-only mode.



## Text object selection
* aw  - a word
* iw  - inner word
* aW - a WORD
* iW
* as - a sentence
* is - inner sentence
* ap - a paragraph
* ip - inner paragraph
* a] - a [] block
* a[
* i[ - inner [] block
* i]
* a) - a block
* a(
* ab
* i( - inner block
* i)
* ib
* a> - a <> block 
* a<
* i> - inner <> block 
* i<
* at - a tag block
* it - inner tag block
* a} - a Block
* a{
* aB
* i{ - inner Block
* i}
* iB
* a" - a quoted string
* a'
* a`
* i"
* i'
* i`

one time command in insert mode
